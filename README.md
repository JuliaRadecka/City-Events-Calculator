# City Events Calculator (v1)

## What it is
A first UI iteration of the "Калькулятор оценки мероприятий".

## Folder structure (flat, as requested)
- app.py
- requirements.txt
- Калькулятор_оценки_мероприятий_по_городам.xlsx

## Run
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Notes
- v1 focuses on UI skeleton: render the Excel-like table and allow editing of input columns.
- Full 1-to-1 calculation (blue cells) will be implemented in v2.
